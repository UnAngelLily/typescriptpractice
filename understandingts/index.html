Core Types in Typescript

number (1, 5.3, -10) = All numbers, no differentiation between intergres or floats
string ('Hi', "Hi", `Hi`) = All text values
boolean (true, false) = Just these two, no "truthy" or "falsy" values
Typescript adds extra steps to develop as a check. Will compile in to javascript.
Make sure that app.ts and app.js are not open at the same time as it may cause duplications. They are viewed as the same file and may cause errors
